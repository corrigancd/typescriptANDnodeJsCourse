## Using typescript and typescript compiler
1. With typescript
Globally install typescript:
```npm install -g typescript```
Then to compile we can use:
```tsc Project.ts```
Then to execute code we can use:
```node Project.js```

2. With ts-node
Globally install ts-node: 
```npm i -g ts-node```
Then to run we can use:
```ts-node Project.ts```
